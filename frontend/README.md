# Expense Tracker (Full-Stack Web Application)

## Overview
This project is a full-stack expense tracking system built using:
- Frontend: React
- Backend: Node.js with Express
- Database: MySQL

It allows users to create, view, update, and delete expenses, categorize them, and generate filtered reports by category.

The backend is deployed on a Google Cloud VM so it can be accessed publicly.


## How to Run (Deployed Version)

The backend is deployed on Google Cloud VM and is already running.
Backend url: http://34.63.225.128:8081/

To run the application, the frontend is started and connects to the deployed backend on Google Cloud.
